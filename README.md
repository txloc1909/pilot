# Pilot

Personal and opinionated coding agent harness. Entirely for author's personal
use.

Aims to be self-improving; I should be able to use `pilot` to develop & improve
`pilot` itself

Start out as a Python port of [pi](pi.dev) v0.72.0

The name `pilot` has dual meaning: 
- Reference to original `pi` project
- `pilot` is actually in-charge
  - Because I personally engineer its orchestration to fit my unique needs
  - Not `copilot` - staying on the sideline.
